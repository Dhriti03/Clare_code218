A Quick Insight


SOME COOL IMPORTANT THINGS PROGRAMMERS SHOULD KNOW!!!

1. Source Control
Creates a warehouse to allow the software engineer to easily share code over a network and the internet
2. Logging
Logging is the act of keeping a log. ... This relieves software developers of having to design and code their own ad hoc logging systems
3. Interfaces (Program to the Interface)
In order to make an application more modular and make the application code easier to refactor, you should always program to the interface.
4. Unit Testing
The automated tests help greatly in teams of multiple software engineers. When the software engineer who wrote the code creates the tests, he makes sure that any other software engineer that comes along to make changes is unlikely to break the code without the test failing
5.Design Patterns
Design patters are the recording of the triumphs that software engineers have encountered in the software world. They are consistently implemented methods of solving common problems we software engineers encounter every day.
